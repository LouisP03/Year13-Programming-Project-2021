Interactive Drawing Application

How to use this app
Use the top slider to adjust the width of your brush. The three sliders below it adjust the amount of red, green, and blue (respectively) in the final selected colour. Use the Reset Button above to reset the canvas. Use the Reset Chat button to reset the chat. To use the colour dropper next to the sliders, simply click on it. Hover over a colour on your canvas to automatically select that colour.
How this app works
This app uses the WebSocket protocol to connect users to a central server. When each user/client updates the canvas or the chat, data is sent as events to the server, which has custom event handlers to broadcast the change data to every user that is connected. The WebSocket protocol provides full-duplex communication over a single TCP connection and changes are broadcast in real time to each user.
